# steamer
A Simple Timer with No Extra Features
![Screenshot]screenshot.png)
# How to compile
1. Install C Compiler, make and ncurses library
2. `git clone https://github.com/andynolife/steamer.git/`
3. `cd steamer`
4. `make`
5. `./steamer`
# Why is it called a steamer?
Simple > S,
Timer > Teamer
= Steamer!
